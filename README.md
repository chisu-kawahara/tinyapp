# TinyApp Project

TinyApp is a full stack web app built with Node and Express that lets users shorten long URLs.

## Dependencies

- Node.js
- Express
- EJS
- bcryptjs
- cookie-session

## Getting Started

- Install all dependencies with `npm install`.
- Start the server with `node express_server.js`.
